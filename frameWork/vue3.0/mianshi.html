# 问题总结

# methods 和 computed 的区别

  1. computed 是有缓存的，只会在数据发生更新的时候触发一次，之后会一直使用缓存。
  2. 二者调用方式不同，methods 需要执行得到结果，computed 直接以对象属性形式调用。
  3. computed 可以通过 get/set 劫持数据。

# v-if 和 v-for

v-if 不建议与 v-for 一起使用

# v-if 和 v-show

  1. v-if 只有在条件成功后才会渲染，v-show 不管条件如何都会渲染，只是 display 的显隐切换。
  2. v-show 只编译一次，而 v-if 不停的新建销毁。所以在多次切换条件的情况下使用 v-show。
